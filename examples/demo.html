How do I implement dark mode with Tailwind v4?

Assistant 2:30 PM

To implement dark mode in Tailwind v4, you can use the @theme directive with dark mode variants.

Here's an example:

@theme {
  --color-surface-default: oklch(98% 0 0);
  
  @media (prefers-color-scheme: dark) {
    --color-surface-default: oklch(20% 0 0);
  }
}
